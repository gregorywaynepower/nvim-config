## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:gregorywaynepower/nvim-config ~/.config/gregorywaynepower/nvim-config
NVIM_APPNAME=gregorywaynepower/nvim-config/ nvim --headless +"PackerSync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=gregorywaynepower/nvim-config/ nvim
```
